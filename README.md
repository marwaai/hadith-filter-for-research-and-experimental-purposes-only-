# marwaai

ABKH: Hadith Filtering & Research Tool

---

## Description
This repository contains tools to **scrape, filter, and experiment with Hadith datasets**, including potentially weak or fabricated Hadiths.  
It is **intended for research and experimental purposes only** and **should not be used for religious rulings or guidance**.

---

## Files in the Repository

- `clean.py`  
  Scripts for cleaning and preprocessing the Hadith data.

- `logistic_model.py`  
  Implementation of a logistic regression model for filtering/classifying Hadiths.

- `scrabing_dorar_fake_hadith.py`  
  Script to scrape Hadith data from Dorar and identify potentially weak/fake Hadiths.

- `scrabing_islamway_fake_hadith.py`  
  Script to scrape Hadith data from Islamway and identify potentially weak/fake Hadiths.

- `requriment.txt`  
  Required Python packages to run the scripts.

---

## Usage

1. Install required packages:
pip install -r requriment.txt

---
## Files

- `clean.py` – Preprocesses and cleans Hadith datasets for modeling.  
- `logistic_model.py` – Runs logistic regression model to filter/classify Hadiths.  
- `scrabing_dorar_fake_hadith.py` – Scrapes Hadith data from Dorar and detects potential weak/fake Hadiths.  
- `scrabing_islamway_fake_hadith.py` – Scrapes Hadith data from Islamway and detects potential weak/fake Hadiths.  
- `requriment.txt` – Lists all Python packages required to run the scripts.
